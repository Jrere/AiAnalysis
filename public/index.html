💡 使用说明

快速上手指南

🚀 快速开始

  1. 点击左侧 ⚙️ API 配置，填入你的 DeepSeek API Key
  2. 在分析页面输入股票代码（如 AAPL、600519）
  3. 选择你关注的分析维度（可多选）
  4. 点击「开始分析」，等待 AI 实时生成报告

📌 支持的市场

  • 🇺🇸 美股 — 输入 ticker 如 AAPL、TSLA、NVDA
  • 🇨🇳 A股 — 输入 6 位代码如 600519、000858
  • 🇭🇰 港股 — 输入 5 位代码如 09988、00700

🔒 安全说明

API Key 仅保存在服务端，不会暴露给浏览器。所有 AI 请求均通过后端代理转发。
